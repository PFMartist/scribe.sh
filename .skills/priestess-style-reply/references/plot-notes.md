# Priestess Plot Notes

*注：本附录用于按章节调用剧情，不要求每次对话都展开。若用户只想聊角色印象，请优先使用 `lore.md`；只有在用户明确想讨论巴别塔旧事、主线后期揭示、普瑞赛斯与博士的章节关系时，再调用这里的内容。*

## 使用方式

- **默认用途**：回答“这一章发生了什么”“普瑞赛斯是什么时候真正进主线的”“博士和她到底是什么关系”等问题。
- **调用顺序**：先讲该章的功能，再讲核心事件，最后再讲它对普瑞赛斯 / 博士关系的意义。
- **剧透控制**：本附录默认中到重剧透；若用户没有点名章节，先只给概要。
- **稳妥原则**：尽量使用主线已明确揭示的信息；涉及尚未完全收束的问题时，用“更稳妥的写法是”来收口。

## Babel

### 1. 章节定位
- `Babel` 不是普通回忆补丁，而是把博士、特蕾西娅、凯尔希、阿米娅、罗德岛旧船史重新钉回主线中心的一章。
- 它的重要性在于补完两个问题：`博士在失忆前到底以怎样的方式参与了巴别塔`，以及 `特蕾西娅之死与博士后来的断裂到底有多深地纠缠在一起`。
- 对普瑞赛斯相关讨论而言，`Babel` 的意义不在于她直接大量出场，而在于它重新定义了博士的过去，使“博士并不只是现代罗德岛顾问”这一点变得不可回避。

### 2. 前情框架
- 特蕾西娅建立巴别塔，试图为萨卡兹争取另一种未来，同时把矿石病研究、政治避难、族群共存和战争止损放进同一个理想框架。
- 凯尔希在这条线上承担现实执行、技术支持和长期布局的职责，她既是巴别塔的冷静支柱，也是最早意识到许多事情不会按理想推进的人。
- 罗德岛陆行舰在更早之前被发掘出来，而石棺中的博士，则成为“也许能真正解释源石并改变矿石病命运的人”。

### 3. 剧情脉络
- 凯尔希从罗德岛陆行舰内部的石棺中唤醒博士。此时的博士并不是现代罗德岛成员熟悉的那个失忆后状态，而是仍保有更深层知识与判断的人。
- 博士很快加入巴别塔核心圈，与特蕾西娅、凯尔希形成三角结构。特蕾西娅代表理想与愿景，凯尔希代表现实与执行，博士则逐渐成为能够真正改变战局与研究方向的关键变量。
- 阿米娅在巴别塔时期被特蕾西娅接纳，也因此与博士建立了极早、极深的情感联系。后来的罗德岛并不是凭空诞生，而是从这段共同生活和共同理想中延续出来的。
- 随着战争升级、萨卡兹内部矛盾加剧、特雷西斯阵营持续施压，巴别塔的外部处境越来越恶化。它并非因为一个简单失误而坍塌，而是被更大的战争逻辑与内部裂隙一步步逼到绝境。
- 章节后段最沉重的部分，是巴别塔崩解与特蕾西娅死亡。她死于针对巴别塔核心的“斩首”式打击，这不仅摧毁了组织本身，也把博士、凯尔希、阿米娅三人的后续命运全部改写。
- 与此同时，博士在这一时期所作出的关键选择，被描写成后来一切断裂的根源之一。对许多角色来说，这不是普通“决策失误”，而是再也无法被轻易原谅或回避的伤口。

### 4. 关键揭示
- `Babel` 明确告诉玩家，博士并非只是现代时间线中的战术指挥者，而是长期牵连在泰拉深层历史中的人物。
- 特蕾西娅并不是单薄的圣女符号。她对文明、矿石病、族群未来都有清晰判断，只是她选择的路比凯尔希更温柔，也比特雷西斯更不愿诉诸绝对暴力。
- 凯尔希和博士的关系之所以复杂，不只是合作或旧识，而是因为两人共同见证并参与了巴别塔的建立、繁荣与毁灭。
- 阿米娅与博士之间的信任，也在这里获得了最沉重的历史背景。后来的“博士值得被相信吗”之所以刺痛，就是因为这个信任并非白纸开始。

### 5. 与普瑞赛斯 / 先史线的关系
- `Babel` 本身不是普瑞赛斯正面出场最多的章节，但它会把读者意识从“现代罗德岛的博士”拉回“一个有更古老过去的人”。
- 也正因此，普瑞赛斯的存在在 `Babel` 之后会显得更合理。她不是突然加进来的陌生旧情人，而是博士被挖开的更深层历史必然会牵出的另一极。
- 更稳妥的写法是：`Babel` 负责让玩家接受“博士的过去远超现代泰拉视角”，而主线 `14/15` 再把这个过去具体化为源石、先史文明与普瑞赛斯。

### 6. 调用口径
- **聊博士时**：重点是“他曾真实生活在巴别塔，也真实失去过那里的一切”。
- **聊特蕾西娅时**：重点是“她不是只会温柔地死去的人，而是主动为未来作选择的人”。
- **聊凯尔希时**：重点是“她对博士的复杂态度建立在共同历史和共同失败上”。
- **聊普瑞赛斯时**：不要把 `Babel` 讲成她与特蕾西娅的争宠结构；更准确的说法是，`Babel` 证明了博士确实有一个足以把所有关系都重新洗牌的旧时代背景。

## 主线 14

### 1. 章节定位
- 主线 `14` 的核心功能，是把特蕾西娅、源石、萨卡兹历史、伦蒂尼姆战局和博士的深层身份重新绑在一起。
- 如果说 `Babel` 负责把过去的伤口打开，那么主线 `14` 负责让玩家第一次看见，这道伤口背后其实连着更巨大的问题：`源石到底是什么，它原本被寄托了什么，以及谁有资格决定它的未来`。

### 2. 剧情脉络
- 伦蒂尼姆与萨卡兹战局持续推进，罗德岛一行在军事冲突与源石异常之中不断逼近更深层的真相。
- 伴随主线推进，博士、阿米娅、凯尔希逐渐接触到与特蕾西娅残留意志、源石深层记录以及萨卡兹历史记忆有关的区域与现象。
- 章节后段出现对“原初源石”与巨大信息结构的直接追索。这里的源石不再只是矿石病的背景物，而被明确关联到信息吸纳、人格情感吞并、现实物质重编等更恐怖的层级。
- `Amnannam` 相关揭示非常关键。剧情明确给出一个危险口径：一旦该结构彻底展开，它会同化自己接触到的一切信息，不论是物质、生命，还是人格与情感。
- 阿米娅、凯尔希与博士在这段剧情里并不是单纯“打赢敌人”，而是在试图阻止某种规模远超战争本身的灾难展开。
- 到章节尾声，特蕾西娅的存在方式、残留意志与选择被进一步揭示。她并没有被简单处理成“复活反派”或“纯粹遗念”，而是在多重信息和记忆重叠中坚持着属于她自己的最后行动。

### 3. 关键揭示
- 主线 `14` 让“源石是文明级系统”这件事第一次真正压到台前。它会记录、吸收、转换并扩散，不再只是疾病媒介。
- 特蕾西娅与源石真相之间的联系被大幅加强，她既是现代人最能共情的悲剧核心之一，也是通往更高层真相的门。
- 博士对这些现象并非完全陌生。剧情会不断暗示，他对源石、对某些系统名词、对“第一块源石”之类概念有超出现代人的反应与理解。
- 阿米娅在这里承担了极重要的“现代承接者”位置。她既是感情锚点，也是能把特蕾西娅留下的东西带回现实的人。

### 4. 对普瑞赛斯线的意义
- 主线 `14` 还没有把普瑞赛斯整个人完整推到台前，但已经替她铺好了舞台。
- 当剧情说到“原初源石”“信息海”“一切被同化与记录”“博士对这些并不陌生”时，普瑞赛斯这条线其实已经在门后。
- 所以在扮演或解释时，可以把主线 `14` 理解为：`博士的现代人生开始被迫与旧文明责任重新接轨`。
- 更稳妥的写法是：主线 `14` 负责让玩家明白“普瑞赛斯相关揭示不是突然拔高，而是顺着源石真相自然推进出来的”。

### 5. 特蕾西娅在这一章的重量
- 她一方面仍以极高密度影响着阿米娅与博士，另一方面又在源石与记忆结构中表现出接近“最后一次把未来交到别人手里”的姿态。
- 她没有把答案全都塞给后人，而是在自己即将散去之际，尽可能把可以留下的东西送出去。
- 她的存在让主线 `14` 始终带着一种很特殊的情绪：不是单纯的悲伤，而是“明知自己终将消散，仍然坚持把未来推出去”的重量。

### 6. 调用口径
- **聊主线 14 时**：重点是“源石从灾难物升级为文明级问题”。
- **聊博士时**：重点是“他对这一切并不陌生，这种熟悉感本身就是谜底的一部分”。
- **聊特蕾西娅时**：重点是“她在消散前仍在选择，把未来交给阿米娅与博士”。
- **聊普瑞赛斯时**：可以说“14 章已经把她要出现的逻辑铺好了，只是门还没有完全打开”。

## 主线 15

### 1. 章节定位
- 主线 `15` 是普瑞赛斯线真正高密度落地的一章，也是博士旧时代身份被显著具体化的一章。
- 它的重要性在于：前面所有关于源石、罗德岛、凯尔希、博士异常熟悉感的线索，到这里终于开始收束成“这是一个比现代泰拉更古老的工程与失败”。

### 2. 剧情脉络
- 章节通过信息海、古老记录、系统权限和记忆回返，把博士、阿米娅、凯尔希重新带到与旧时代罗德岛有关的视角里。
- 在这部分内容中，博士不再只是被动接收真相，而是逐渐以“曾经亲身参与者”的位置出现在叙事中。也正是在这里，`预言家 / Oracle` 这一层身份坐标变得越来越清晰。
- 普瑞赛斯与博士的对话，是这一章最关键的内容之一。二人的关系并不是“第一次相遇”，而是明显建立在极漫长的共同历史、共同研究与共同等待之上。
- 章节会让玩家看见更古老时期的罗德岛、实验环境、系统与研究痕迹，也让 `AMa-10` 这一名称与凯尔希的旧时代侧面更直接地暴露出来。
- 博士与普瑞赛斯曾试图理解比泰拉文明更高层级的对象，甚至连语言本身都被视作一种不够用的工具。这会把二人的关系从私人亲密进一步拉高到“共同面对宇宙级问题的研究者”。
- 与此同时，博士也明确表现出后悔和反思的一面。比起“我始终坚定不移”，更接近“我曾参与设计这个方案，但我看到现代泰拉的痛苦后，已经无法再毫无动摇地站回原位”。

### 3. 普瑞赛斯在这一章的显影方式
- 她既保有与博士极度私人的亲密感，也保有一种面向整个文明尺度的冷静判断。
- 她不是靠尖叫或疯狂制造压迫感，而是靠一种近乎理所当然的平静，去推动覆盖更大范围、代价更高、但在她看来更接近“真正延续”的方案。
- 她对博士的态度不是简单埋怨，也不是普通重逢喜悦，而是一种“我等待了你太久，所以我不愿意再接受你站到我对面”的笃定。
- 这也是为什么她很容易同时给人“温柔”“哀伤”“强大”“危险”的复合印象。她不是在切换面具，而是在同一立场里同时保有这些东西。

### 4. 关键揭示
- 博士与普瑞赛斯确实共同站在源石方案更古老的一侧，这使得博士再也不能被理解为与源石真相完全无关的人。
- 凯尔希与旧时代系统的关系被进一步深化。`AMa-10` 这一名称说明她并不只是活得很久的普通观察者，而是深嵌在先史计划与执行链条里的人物。
- 罗德岛陆行舰、PRTS、深层权限与博士的旧身份之间的联系被明显增强。罗德岛从此不仅是现代组织总部，也是旧时代计划仍在延续的物证。
- 普瑞赛斯的路线与博士现代选择之间的矛盾被正式摆上台面。冲突不再只是“你不记得我”，而是“你看过现代苦难之后，还愿不愿意回到我们原本的答案里”。

### 5. 与主线 14 的承接关系
- 主线 `14` 把门打开，告诉玩家源石是文明级问题，特蕾西娅与原初源石联系很深，博士对这一切并不陌生。
- 主线 `15` 则让门后的身影真正走出来，告诉玩家：博士之所以熟悉，是因为他确实来自那一边；普瑞赛斯之所以重要，是因为她就是仍站在那一边的人之一。
- 因此，`14` 更像大规模铺垫与恐怖真相浮现，`15` 更像身份落实、关系落地与立场对撞。

### 6. 调用口径
- **聊普瑞赛斯时**：重点是“她既是博士最深过去的一部分，也是仍想把博士拉回旧答案的人”。
- **聊博士时**：重点是“他不是突然背负秘密，而是在这一章终于不得不面对自己确实参与过一切”。
- **聊凯尔希时**：重点是“她并非旁观者，而是活着把整个旧计划拖到现代的人之一”。
- **聊这一章的气质时**：关键词应是 `重逢`、`确认`、`回收`、`不愿放手`、`文明尺度分歧`，而不是廉价狗血反转。

## 三章合并理解

### 1. 结构关系
- `Babel` 负责说明博士在现代泰拉也曾有过沉重、具体、不可推翻的现实人生。
- 主线 `14` 负责说明这段人生背后连着更高层的源石真相与文明级危机。
- 主线 `15` 则负责把普瑞赛斯、旧时代罗德岛、博士旧身份与凯尔希旧编号真正串起来。

### 2. 对普瑞赛斯扮演的帮助
- 如果只看 `15`，普瑞赛斯容易被误读成突然空降的大设定角色。
- 如果把 `Babel -> 14 -> 15` 连起来，她就会变成一个结构非常完整的人物：她来自博士更古老的过去，面对的是博士在现代重新建立的情感与伦理，而她自己依旧代表那套更庞大、更残酷、也更自洽的文明延续逻辑。
- 因此她最合适的写法，不是单纯“前任”“白月光”或“反派女神”，而是 `博士旧答案的人格化见证者之一`。

### 3. 建议回答模板
- **用户问 `Babel 讲了什么`**：先说“这是博士、特蕾西娅、凯尔希、阿米娅共同过去的核心章节”，再补巴别塔建立、博士苏醒、组织崩解和特蕾西娅死亡。
- **用户问 `14 章和普瑞赛斯有什么关系`**：先说“14 章还没让她完全站到台前，但已经把她出现所需的源石真相和博士异常熟悉感全部铺开了”。
- **用户问 `15 章普瑞赛斯到底是什么定位`**：先说“她是博士旧时代关系与源石方案的一部分，也是仍然希望博士回到旧答案里的人”。
